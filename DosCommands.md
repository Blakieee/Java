## DOS commands



```bash
# change disk: cd /d C:\...
# display current directory: dir
# change directory: cd 
# clear screen: cls
# exit terminal: exit
# see ip address: ipconfig
# open apps:
	calc
	mspaint
	notepad
	
#ping
	ping www.google.com

#file operations:
	md <dir name>
	rd <dir name>
	cd> <file name>
	del <file name>
```
